# Scam of the Day (Web + API)

A senior-friendly "Scam of the Day" web app that shows real scam alerts from trusted public sources via a backend API.

## Features
- Scam of the Day (by date)
- Yesterday / Today / Tomorrow
- Search (requires /api/scams)
- Offline/demo fallback if backend unavailable

## Requirements
- Node.js 18+ (for backend)
- Any modern browser (for frontend)

## Run locally

### 1) Start backend
```bash
cd backend
npm install
npm start
